mvoauthphpapi
=============

Mobile vikings API php client

To use it in your application, juste add in your composer.json the following lines :

{
    "repositories": [
        {
            "url": "https://github.com/jflefebvre/mvoauthphpapi.git",
            "type": "git"
        }
    ],
    "require": {
	    "jflefebvre/mvoauthphpapi":"dev-master"		
	}
}
